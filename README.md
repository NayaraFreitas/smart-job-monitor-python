> Transformando a busca manual de emprego em inteligência de dados: Automação que reduz em até 95% o tempo gasto na procura por vagas.

## 🚀 Sobre o Projeto
O objetivo desta solução de engenharia de dados e automação (RPA) é monitorar , coletar e centralizar vagas de tecnolgoia em tempo real. 
Buscar vagas manualmente em multiplos sites abaca consumindo horas preciosas da semana. Este projeto busca resolver essa dor automatizando todo o processo de *web scraping*, limpeza e consolidação dos dados. O rôbo é programado para rodar de forma agendada todo começo de semana, entregando um relatório pronto com as melhores oportunidades e têndencias do mercado.

### 💡 Impacto e Resultados
* **Eficiência:** Redução estimada de **85% a 95% no tempo** gasto com busca manual de vagas.
* **Inteligência:** Centralização de dados brutos da web em um único arquivo estruturado para análise de mercado.
